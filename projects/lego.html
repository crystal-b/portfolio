Electronics: LEGO Robot

Group Members: Vagner Domingues, Gabriel Pernas

My group built a robot using the LEGO Mindstorms NXT kit. Our goal was to get the robot to complete an obstacle course using sensors for light, proximity, sound and touch. The real challenge was to build and code the robot without guidance from our professor.

We figured out the basics of the IDE using tutorials in the program and manual. We researched similar projects and YouTube videos for tricky issues like setting individual wheel speeds.

Along the way, we made several modifications to allow the robot more turning room in tight spaces and to make sure the ultrasonic sensor read obstacles at the correct height.

My contribution: I was largely responsible for the rebuild and modifications. I assisted with the research on how to code the various functions and wrote the code for the proximity, reverse movement, audio and touch sensors. I also performed the sensor calibrations and documentation of the project.
