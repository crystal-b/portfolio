Web Redesign: Freestyle Photographics

As part of a multimedia redesign for my Desktop Publishing class, I gave the Freestyle Photographic Supplies website a design overhaul. I focused on using bold images and implementing a broader color scheme.

At the time, Freestyle seemed to suffer from having an unfocused sense of their audience. Much of their marketing was aimed at both young students and established professionals, so I chose to have the website target the younger audience specifically.

Web Design: Morris Luggage

I created this website using the ZURB Foundation 5 framework and additional CSS. The client mentioned that a significant amount of customers use their smartphones to locate the business, so the framework was perfect for a responsive build that will display well on various devices.

Because the content was very simple, I created a one-page, brochure-ware type website. The most advanced features are the interactive map and the fixed navigation, which allows mobile users to jump around with minimal scrolling.
